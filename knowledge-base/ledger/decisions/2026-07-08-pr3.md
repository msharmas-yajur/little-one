# PR #3 — `count` mechanic: revised, then merged

**What was proposed.** The Mechanic's first run: a `count` game type
("how many? tap each one"), schema branch + registry entry + example
(`count-with-me`), first 2-3y/3-4y content.

**What the gates said.** Schema and mechanic-lint passed. The Council
escalated: Guardian + Explorer rejected, four seats said revise; 5/6 seats
independently converged on "the prompt is a quiz with a right answer."

**What the owner did.** Revise-then-merge (owner chose Council-first review,
then ruled on the escalation).

**Revisions adopted (from the seats):**
- Prompt reframed from quiz to invitation: "Let's count the {label} —
  together!" (5/6-seat convergence)
- Grown-up cue restored (Voice Seat): per-round `cue` added to schema,
  renderer, and all five rounds — serve-and-return, the grown-up voices the
  counting, the pause is the turn
- Honest tags (Skills Seat): `critical-thinking` → `connections`

**Owner class rulings (Guardian's two escalation grounds):**
1. **Forward age-bands (2-3y/3-4y) are in scope.** The founding fact fixes
   where the little one is *today*; the schema and roadmap deliberately plan
   forward. Rubric note for COUNCIL.md: forward-band content is not an age
   red-line by itself.
2. **The `count` mechanic is approved** as a class. The owner's merge *is*
   the class decision the charter reserves.

**Findings NOT adopted, and why (calibration signal):**
- Explorer/Guardian's "counting-to-a-target is a fail-state test": reviewed
  against the *code*, an incorrect count is impossible — no numeric answer is
  ever given; taps only count up; repeats are gentle no-ops. The Council
  reviews the YAML artifact, not the renderer — scope limit to remember.
- Voice Seat's strongest form ("the app should not voice the counting"):
  partially adopted. The app keeps its gentle spoken count-words (consistent
  with find-mode speaking labels); the new cues make the grown-up the primary
  counting voice.

**Why (one line).** The mechanic was sound and the code already no-fail; the
copy and the missing grown-up cue were real regressions worth fixing before
they shipped — and the Council caught both.
