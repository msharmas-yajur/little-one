# Idea — an encouraging "cheer" button

Owner idea (2026-07): add a friendly button/CTA that calls out warm encouragement
to the little one — when she gets something right, and gently "otherwise" too.
Pressable by the **child** (cause-and-effect delight, agency) or the **grown-up**
(a reward moment).

## Decision recorded: NOT famous cartoon voices
The original thought was famous cartoon character voices "freely available" online.
We are **not** doing that:
- Famous character voices/clips are protected by **copyright** (recording + character)
  and usually **trademark**; voice actors hold **publicity rights**. "Free download"
  sites are almost always unlicensed rips.
- It's a **public** website for a child — exactly the use rights-holders pursue.
- It breaks the project's own rule: **original only; no copyrighted characters**
  (see `wiki/concepts/design-principles.md`).

## Chosen direction: original mascot + CC0 / original sounds
Invent our **own** friendly "Little One" mascot (original SVG, same style as
`art.js`) that cheers using **original Web-Audio jingles** first (zero files, fully
owned), with the option to add **CC0 / public-domain** clips or a grown-up's own
recorded voice later. Same joy, no third-party IP.

## Must stay forgiving
"Otherwise" (a wrong tap) is still **kind** — a gentle "ooh, try again!" nudge,
never a scold, no fail sound, no score lost. (Design rule: the game never scolds.)

## Hooks
- Cause-and-effect (arc Month 3) and agency (Month 9) when the child presses it.
- Related backlog already noted: **real animal sounds on tap** (`famous-books.md`,
  `storybook-player.md`).
