# How to use raw/ideas

Drop loose ideas, wishes, and observations here as plain markdown. One idea per
file is cleanest, but a running notes file is fine too. Don't worry about format
or polish — this is *raw*. A session will ingest it and compile it into the wiki.

**No names.** Write "the little one," not a real name.

Example things to drop here:
- "She lit up at the duck picture — maybe a whole farm-animal story?"
- "Idea: a bedtime mode that dims the colours after 7pm."
- "When she taps the sun, could it hum a rising note?"

Then, in a Claude session: "Read raw/ideas/<file> and ingest it into the wiki."
